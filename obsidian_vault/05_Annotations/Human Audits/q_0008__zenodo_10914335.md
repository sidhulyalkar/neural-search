## Dataset
**Spike recordings from micro-electrocorticographic arrays**

<p><strong>General description:</strong> This is the sample micro-ECoG recording for a sample animal (rat) for a sample session. Electrode dimensions are 30x30&mu;m, with center-to-center pitch of 400 &mu;m. The array was microfabricated on a flexible, ultra-thin parylene C substrate (4 &mu;m). The array was placed over the rat's right M1 cortex (2✕2 cm craniotomy, with lower left corner 1 mm anterior and lateral to the bregma).</p>
<p>The rat performs spontaneous navigation within a rectangular test arena, with simultaneous recording through the micro-ECoG array.</p>
<p><strong>File type: </strong>Pickle file containing a <em>Python</em> dictionary object.</p>
<p><strong>Variable names:&nbsp;</strong><em>N: </em>number of sorted units,&nbsp;<em>k:</em> number of samples,&nbsp;<em>t:</em> number of time steps</p>
<ul>
<li><strong>unit identifier</strong>: <em>uN (e.g., u0,</em> u1, ... u58)
<ul>
<li><strong>spike_timing_ms</strong> - <em>k</em> x 1: spike timing in milliseconds (with respect to spike troughs)</li>
<li><strong>spike_waveform_mean </strong>- <em>t</em> x 1: mean spike waveform</li>
<li><strong>wave_form </strong>-<strong> </strong><em>t</em> x <em>k</em>: individual spike waveforms for each spike, units in microvolts. 100ms per time step (-1 ms and +3 ms around spike trough)</li>
<li><strong>rat</strong>: animal identifier</li>
<li><strong>ch</strong>: electrode channel</li>
<li><strong>unit</strong>: unit identifier</li>
</ul>
</li>
</ul>
<p><strong>Possible uses:&nbsp;</strong>This can be used to compare with other micro-ECoG recordings. The spikes likely arise from layer I neurons, which lie in proximity to the surface recording microelectrodes.</p>
