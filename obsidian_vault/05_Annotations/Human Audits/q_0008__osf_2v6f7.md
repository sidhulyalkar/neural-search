# Dataset: A Vascular Code for Speed in the Spatial Navigation System

This dataset contains the minimally processed data of the paper "A Vascular Code for Speed
in the Spatial Navigation System", from Cybis Pereira et al. (https://doi.org/10.1016/j.celrep.2025.116791)

The dataset should follow closely what is stated in [BEP 040](https://bids.neuroimaging.io//extensions/beps/bep_040.html), even though this extension proposel is still under active development.

## Derivatives

- `dlc-videos`: Contains labeled videos from the DeepLabCut tracking system used to track the animals during the freely moving fUS imaging experiments.
- `glm-speed`: Contains the outputs of General Linear Model approaches performed for animal speed.
- `glm-angular-speed`: Contains the outputs of General Linear Model approaches performed for head angular speed.
- `decode-speed`: Contains the outputs of decoding models for animal speed.
- `interanimal-decode-speed`: Contains the outputs of decoding models across animals for animal speed.
- `slowwave`: Contains the outputs of slow-wave oscillations analysis (this derivative should be requested through the corresponding author of the paper as it is too heavy to be deposited in Zenodo).

First author, dataset collector and curator:
- Cybis Pereira, Felipe (https://orcid.org/0009-0009-6124-0466)
