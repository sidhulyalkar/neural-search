# Dataset Card Review Checklist

Use this checklist before showing dataset cards to anyone.

## Manual Review Checklist

For each high-priority dataset card, verify:

### Basic Information
- [ ] Is the title correct?
- [ ] Is the source URL correct?
- [ ] Is the source ID correct?

### Scientific Labels
- [ ] Are task labels justified?
- [ ] Are modality labels justified?
- [ ] Are behavior labels justified?
- [ ] Are brain regions verified or clearly marked uncertain?

### Linked Content
- [ ] Are linked papers actually related?

### Quality Indicators
- [ ] Does the readiness score make sense?
- [ ] Are missing fields shown?
- [ ] Are suggested analyses realistic?

### Hallucination Check
- [ ] Is there any hallucinated claim?

### Technical Validation
- [ ] Can the notebook generator run for this dataset type?
